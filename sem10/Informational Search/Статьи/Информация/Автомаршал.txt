Автомаршал — программа распознавания автомобильных номеров.
Применяется для автоматизации учёта проезда машин, проверки их по базам данных и ограничения доступа машин на территорию.

Область применения
контрольно-пропускные пункты предприятий;
платные автостоянки;
парковки у торговых комплексов;
охраняемые и закрытые территории;
жилые комплексы, ТСЖ;
коттеджные посёлки;
автомойки;
автоматический розыск транспорта.

Функции
Автоматическое обнаружение машины в поле зрения камеры.
Распознавание российских автомобильных номеров, соответствующих ГОСТ Р 50577-93.
Сохранение в базе данных информации об автомобиле: номер, фото, дата, время, факт совпадения с записью в какой-либо из баз данных и др.
Оповещение оператора об обнаружении машины: отображение распознанного номера с возможностью корректировки.
Поиск в базе данных обнаруженных машин по различным критериям.
Создание отчетов по обнаруженным машинам: XML, MS Excel, MS Word и др.
Вывод на печать информации об автомобиле.
Создание отчета о работе оператора с программой (регистрация даты, времени включения и выключения программы; изменений баз данных; пароля оператора).
Сохранение фотографии машины без номера.
Обнаружение машины с помощью аппаратных датчиков движения.
Управление внешними устройствами (шлагбаум, автоматические ворота и т. д.).
Взаимодействие с любым весовым оборудованием.
Сохранение изображений в БД, на диск или передача по сети.
Видеозапись проезда автомобиля с камеры.
Отправка отчетов по электронной почте.
Оповещение пользователей по SMS.

Возможности
Разграничение полномочий для работы с программой (администратор, пользователь).
Объединение работы нескольких КПП.
Поддержка до 8 видеоканалов в базовой версии.
Возможность интеграции в системы верхнего уровня (системы безопасности, 1С:Предприятие и т. д.).
Гибкая настройка бизнес-логики.

Технологии
Распознавание автомобильных номеров производится с помощью ядра распознавания RecarKernel (разработка научно-производственной компании «Малленом»)
Поддерживается работа с большинством устройств видеоввода:

Аналоговые видеокамеры (посредством платы видеоввода).
Устройства, поддерживающие технологию DirectShow (цифровые видеокамеры, web-камеры, специализированные платы видеоввода).
IP-камеры: Basler, Axis, Arecont Vision, Corum, MicroDigital и др.Доступна работа с различными СУБД: Firebird, MySQL, Oracle Database, Microsoft SQL Server и др.
В базовой версии используется СУБД Firebird.
В Автомаршале используется клиент-серверная технология:

Использование сетевой СУБД.
Сервер, отвечает за распознавание автомобильных номеров.
Клиентские части позволяют просматривать журнал обнаруженных машин, редактировать пользовательские базы данных машин, просматривать фото и видео с камер, распечатывать отчеты.
Количество клиентских компьютеров не ограничено.
Передача видео по сети между клиентом и сервером.

Система защиты
Для защиты Автомаршала используются аппаратные USB-ключи защиты.

Модификации
Автомаршал в модификации «Дорожный пристав» представляет собой аппаратно-программный комплекс, применяемый для выявления в потоке автомобилей должников по различным платежам и правонарушениям. Используется Федеральной службой судебных приставов, Налоговой инспекцией, ГИБДД и др. Также реализована система использования плагинов, позволяющая расширить и дополнить функциональность системы.
