WebSpellChecker Software — многоязычное программное обеспечение, которое осуществляет поиск ошибок и опечаток как отдельно взятых слов, так и объемных текстов. Сервис основывается на использовании орфографических словарей на 158 языках и подборе правильных вариантов написания. Целевая аудитория охватывает системы управления клиентами, инструменты интернет-маркетинга, сферу образования и государственных учреждений.

История создания
В 2000 году американская компания SpellChecker.net, Inc. создала продукт WebSpellChecker.
В июне 2016 года права на программное обеспечение было продано украинской компании WebSpellchecker LLC.
Команда разработчиков расположена в Харькове.

Продукты и интеграция
WebSpellChecker интегрируется и используется в связке с HTML элементами: input, textarea, iframe и т. д. Также он доступен как плагины для таких популярных текстовых редакторов как CKeditor 4+,TinyMCE4+ и Froala.
В конце мая 2017 года WebSpellChecker Software стало доступно для всех пользователей WordPress. 
А уже в июне 2017 года WebSpellChecker запустил бесплатное Slack приложение для проверки орфографии на 8 языках.WebSpellChecker предоставляет четыре продукта:

SCAYT — проверка орфографии в процессе написания текста позволяет пользователю мгновенно исправлять недочеты. Найденные ошибки или опечатки подчеркиваются красной волнистой линией. Исправление производится при клике на слово и выборе из выпадающего списка рекомендуемых замен.WSC — проверка орфографии в отдельном окне происходит после вставки фрагмента текста в поле и нажатия кнопки «Проверить правописание». Для данного продукта также реализована функциональность для проверки Английской грамматики и Тезаурус.Web API — это список команд и интерфейсов JSON/XML с помощью которых задаются пользовательские настройки для реализации проверки орфографии.
WProofreader —  новый продукт c современным пользовательским интерфейсом, объединяющий в себе функциональность SCAYT и WSC.

Совместимость
Программное обеспечение WebSpellChecker совместимо с платформами Windows и Linux с предустановленным и настроенным GCI-совместимым веб-сервером (IIS или Apache) или JE22 на базе сервера приложений Apache Tomcat или Weblogic.
WebSpellChecker поддерживает следующие браузеры: Internet Explorer 9+, Firefox (последний), Google Chrome (последний) и совместим с Safari (последним), EDGE.

Поддерживаемые языки
WebSpellChecker поддерживает проверку офрографии для 16 языков по умолчанию. Из них для 14 языков  еще доступна проверка грамматики.

Пользователю доступно использование дополнительных 140+ языков и специализированных юридического и медицинского словаря.
