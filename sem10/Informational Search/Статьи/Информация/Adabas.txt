Adabas (от англ. adaptable database system) — система управления базами данных разработки Software AG. Впервые выпущена для мейнфреймов IBM в 1971 году, изначальная модель данных ­— на базе инвертированного индекса, в 1980-е годы дополнена элементами реляционной модели. До взлёта популярности в середине 1980-х годов  реляционных СУБД, была одной из самых продаваемых систем управления базами данных (11% рынка в 1983 году).
По состоянию на 2019 год выпускается также для платформ Linux, Unix и Windows.
Оснащена собственным сильно типизированным языком запросов — Natural. Пример программного кода:
