# Задание

Составить на языке Си текстовый процессор

Процессор должен работать в двух режимах – интерактивном и командном.

Возможности процессора:

* показ статистики файла
* поиск по началу слова/всему слову/концу слова
* задание размера лимита файла
* показ определенной строки файла 
* вставка на определенную позицию
* удаление части символов из строки с определенной позиции.
