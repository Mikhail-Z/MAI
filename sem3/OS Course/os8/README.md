# Цель работы
## Целью является приобретение практических навыков в:
*	Создание мобильных приложений
*	Использовании системных вызовов мобильных ОС

# Задание
Требуется создать простое мобильное приложение, используя различные системные вызовы. Необходимо продемонстрировать работу с компонентами мобильной ОС на выбор студента:
1.	iOs
2.	Android
3.	Windows phone
В конечном итоге, программа должна состоять из следующих частей:
*	Программный код, отвечающий за отображение приложения;
*	Программа обеспечивающая взаимодействие пользователя, мобильной ОС и отображения.
