# ЗАДАНИЕ

Необходимо спроектировать и запрограммировать на языке C++ класс-контейнер первого уровня (*очередь*), содержащий 3 разных типа автомобилей, имеющих разные характеристики.
Классы должны удовлетворять следующим правилам:
*	Требования к классу фигуры аналогичны требованиям из лабораторной работы 1.
*	Классы авто должны иметь переопределенный оператор вывода в поток std::ostream (<<). Оператор должен распечатывать параметры авто (тип, ост. хар-ки).
*	Классы авто должны иметь переопределенный оператор ввода фигуры из потока std::istream (>>). Оператор должен вводить основные параметры авто (тип, хар-ки).
*	Классы авто должны иметь операторы копирования (=).
*	Классы авто должны иметь операторы сравнения с такими же авто (==).
*	Класс-контейнер должен соджержать объекты авто “по значению” (не по ссылке).
*	Класс-контейнер должен иметь метод по добавлению авто в контейнер.
*	Класс-контейнер должен иметь методы по получению авто из контейнера (опеределяется структурой контейнера).
*	Класс-контейнер должен иметь метод по удалению авто из контейнера (опеределяется структурой контейнера).
*	Класс-контейнер должен иметь перегруженный оператор по выводу контейнера в поток std::ostream (<<).
*	Класс-контейнер должен иметь деструктор, удаляющий все элементы контейнера.
*	Классы должны быть расположенны в раздельных файлах: отдельно заголовки (.h), отдельно описание методов (.cpp).

Нельзя использовать:
*	Стандартные контейнеры std.
*	Шаблоны (template).
*	Различные варианты умных указателей (shared_ptr, weak_ptr).

Программа должна позволять:
*	Вводить произвольное количество автомобилей и добавлять их в контейнер.
*	Распечатывать содержимое контейнера.
*	Удалять автомобили из контейнера.
