* Список
Каждым элементом контеинера, в свою, является динамическои структурои данных одного из следующих видов (Контейнер 2-го уровня):
* Очередь

Нельзя использовать:
*	Стандартные контейнеры std.

Программа должна позволять:
*	Вводить произвольное количество авто и добавлять их в контейнер.
*	Распечатывать содержимое контейнера (1-го и 2-го уровня).
*	Удалять авто из контейнера по критериям
