# Основы построений фотореалистичных изображений

Используя результаты Л.Р.No2, аппроксимировать заданное тело
выпуклым многогранником.

Точность аппроксимации задается пользователем. Обеспечить
возможность вращения и масштабирования многогранника и
удаление невидимых линий и поверхностей.

Реализовать простую модель закраски для случая одного источника
света. Параметры освещения и отражающие свойства материала
задаются пользователем в диалоговом режиме.
