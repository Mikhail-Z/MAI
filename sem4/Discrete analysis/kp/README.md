# Параллельная внешняя сортировка слиянием

##Формат файлов:
###Файл с ключами: бинарный файл
uint64_t [n] --- количество ключей в файле

uint64_t [l] --- длина ключей в байтах

(uint8_t * l) * n --- n ключей по l байт
###Файл со значениями: бинарный файл

uint64_t [n] --- количество значений в файле

uint64_t [l] --- длина значений в байтах

(uint8_t * l) * n --- n значений по l байт
