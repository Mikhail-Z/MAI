# Многопоточная реализация алгоритма Беллмана-Форда

## Данный алгоритм ищет кратчайший путь между двумя вершинами в ориентированном взвешенном графе

### Требуется установленные компоненты OpenMP

В папке с исполняемым файлом должен быть файл с количеством вершин и рёбер, стартовой и конечной вершинами в первой строке
и списком рёбер вида <start, finish, weight> в последующих строчках. Файл должен иметь название 'sample.gr'.
