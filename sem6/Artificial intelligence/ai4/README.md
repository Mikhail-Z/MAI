# Когнитивные сервисы Microsoft

## Необходимо с помощью когнитивных сервисов анализа изображений провести анализ аккаунтов вконтакте и сделать выводы в графическом виде

### Вариант: Взять аккаунт пользователя, у которого более 100 друзей, и проанализировать их на предмет зависимости количества котиков на фотографиях 
от возраста, пола и количества друзей.

### Внимание. Перед использованием зарегиструйтесь на сервисе Microsoft Azure и приобретите Vision Api. Полученный ключ вставьте в `apikey = ''`.
Также предварительно создайте приложение в Вконтакте в меню "Мои Приложения". Полученный идентификатор приложения вставьте в `APP_ID = None`
